# PHP8.1+ 通用工具包

> 这是一个基于PHP8.1+特性开发的模块化通用工具包，提供了数组处理、字符串处理、时间处理、加解密、消息体、IP地址处理、地理计算、全局辅助方法等功能。支持对象和静态两种调用方式，兼容进程和协程环境。

## 安装

```bash
composer require cdyun/php-tool
```
